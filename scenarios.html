// Scenario 01 — Data Transmission
⚡ Fibre Optic — Mode Number Calculation
A fibre optic cable supports multiple transmission modes. The approximate number of modes N is given by the formula below, where d is core diameter, n₁ is core refractive index, n₂ is cladding refractive index, and λ is the wavelength of light.
N  =  [ (π · d · √(n₁² − n₂²)) / λ ]²  ÷  2
dbl-click // Q01
Compute the number of modes for a fibre with:
core diameter d = 20 μm  |  n₁ = 1.5  |  n₂ = 1.48  |  wavelength λ = 1300 nm (typical single-mode laser).
✦ Step 1 — Numerical Aperture (NA):
   NA = √(n₁² − n₂²) = √(2.25 − 2.1904) = √0.0596 ≈ 0.2441

✦ Step 2 — V-number (normalised frequency):
   V = (π · d · NA) / λ = (π × 20×10⁻⁶ × 0.2441) / (1300×10⁻⁹)
   V ≈ (3.1416 × 0.000020 × 0.2441) / 0.0000013 ≈ 11.77

✦ Step 3 — Number of modes:
   N = V² / 2 = (11.77)² / 2 ≈ 138.5 / 2 ≈ ~69 modes

✦ Key insight: For single-mode operation, V < 2.405. At V ≈ 11.77 this fibre is multi-mode. To achieve single-mode, reduce d or increase λ.

Bubble sort repeatedly compares adjacent elements and swaps them if out of order. Each full pass "bubbles" the largest unsorted value to its final position at the end.

Initial Array:  [ 25, 34, 98, 7, 41, 19, 5 ]
